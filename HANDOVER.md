## What yAgents is
A free, open-source, public tool that visualizes Claude Code AI agents as a live pixel-art world. It reads Claude Code's JSONL transcripts (`~/.claude/projects/**/*.jsonl`) and renders, on an HTML canvas, ONE pannable archipelago: each project folder = an organic themed ISLAND (scholar's camp / dig site / farmstead, **theme per folder**) that grows with its session count; each session = a station on it where the crowned main agent works (scribe desk / ore face / crop plot); subagents join the same job; idle agents from any session loaf in the island's single shared lounge; islands sit on a dark sea, connected by stepping-stone causeways.

## Architecture (pipeline)
`adapters` (FolderPicker / LocalServer / Demo) → `parser/transcript.ts` (parseLine) → `parser/session.ts` (SessionTracker state machine) → `state/officeStore.ts` (live OfficeView, 5-min expiry / 30-min waiting window) → `layout/layoutEngine.ts` (WorldPlan: spiral-placed island Regions→stations+shared lounge+landmark+gateway→seats, terrain math in `layout/terrain.ts`) → `render/renderer.ts` (sea, causeways, per-island offscreen terrain cache; delegates drawRegionBase/drawRegionLive/drawCharacter/drawEffects to the active `Skin`) + `ui/detailPanel.ts` overlay. Pure, deterministic, **no `Math.random`** (seed from time/hash — required for the procedural world).

Key contracts: `render/theme.ts` (`themeFor`, `poseFor`, `Pose`), `render/atlas.ts` (`Theme`, `THEME_TILES`), `layout/layoutEngine.ts` (`Station`/`Region`/`Seat`/`WorldPlan`), `layout/terrain.ts` (`inBlob`/`packCell`/`quadrantSources` — blob membership is monotonic in R, a hard invariant). `agentKey` = `tableKey` for mains, `tableKey + '#' + subId` for subs (detail panel splits on `#`).
