# yAgents

A live pixel-art office for your Claude Code agents. Every folder is a room, every session a table — main agent at the head, subagents around it. Waiting agents grab a coffee in the kitchen.

**Live:** https://yaalw.github.io/yAgents/ (Chromium browsers; pick your `~/.claude/projects` folder)

**Local (any browser):** `node server/src/index.js` after `npm install && npm run build`

**Demo:** append `?demo=1` to either URL.
